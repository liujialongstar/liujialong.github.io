Markdown语法

发表于 2019-12-26 更新于 2021-01-29 分类于 Markdown

Markdown 致力于使阅读和创作文档变得容易。

Markdown 视可读性为最高原则，语法完全由标点符号组成。

Markdown 的目标是易于阅读、创作和编辑文章。HTML 是发布形式；Markdown 是创作形式

代码块

要在Markdown中插入代码块，只需要将每一行都缩进4个空格或1个制表符。

此外使用一对 ` 可以插入行内代码，使用一对 ``` 可以插入代码块：

`插入行内代码`
